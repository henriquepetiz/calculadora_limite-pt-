Exercício de Python em Programação de Computadores - Aula 3 parte 5

Calculadora de Limite de Crédito

Este é um script em Python que calcula o limite de crédito de um cliente baseado no seu saldo médio mensal. O programa usa diferentes porcentagens dependendo da faixa de valor do saldo.
Como funciona

O cálculo do limite segue estas regras:

    Saldo abaixo de 500: Sem limite disponível.

    Saldo entre 500 e 1000: 8% do saldo médio.

    Saldo acima de 1000: 14% do saldo médio.

Funcionalidades

    Validação de entrada: O programa não trava se você digitar letras.

    Repetição: Ele continua pedindo o valor até que um número válido seja digitado.

    Formatação: O resultado final exibe apenas duas casas decimais.

Como usar

    Tenha o Python instalado no seu computador.

    Baixe o arquivo main.py.

    No terminal, execute: python main.py

    Digite o saldo médio quando solicitado.
